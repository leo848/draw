# draw
## A website to draw using HTML canvas. Can even be useful.
**Features**
+ Size canvas
+ Draw
+ Change color with **CSS** attributes
+ Change pen size
+ Clear the canvas without losing pen color or size
+ See the image
+ Download the image

That's __currently__ everything you can do.

**How to use**
+ Set the canvas size
+ Draw with your mouse
+ Type '#', 'c'
